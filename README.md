# Calendar Booking System API

A Node.js backend service built with Express and Sequelize to manage user meetings while preventing overlapping schedules.

## Features
- User management
- Conflict detection for meetings
- Date and user-based filtering
- Clean modular architecture

## Setup
1. Install dependencies: npm install
2. Configure .env
3. Run migrations: npx sequelize-cli db:migrate
4. Start server: npm start
